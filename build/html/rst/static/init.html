Initialize API¶

Initialize layers¶

RandomNormal_¶

Examples:

input_layer = vqnet.Input([3,17,17])
conv2d_layer = vqnet.Conv2D(input_layer, 16, [3,3], [2,2])
a = conv2d_layer.params[0].getdata()
vqnet.RandomNormal_(conv2d_layer,m=0.0, s=0.1, seed=1234)
b = conv2d_layer.params[0].getdata()
print(a,b)
